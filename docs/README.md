# Afriex SDK Documentation

This folder contains the documentation website for the Afriex SDK, built with [Nextra](https://nextra.site) and Next.js App Router.

## Getting Started

1.  Running this project requires the root dependencies to be installed:
    ```bash
    pnpm install
    ```

2.  Start the development server:
    ```bash
    pnpm dev
    ```

## Structure

- `app/`: Next.js App Router pages and layouts.
- `app/layout.tsx`: Root layout with Nextra configuration.
- `app/page.mdx`: Landing page content.
- `next.config.mjs`: Nextra plugin configuration.
- `theme.config.tsx`: Nextra theme customization.
