# Wallets

The Wallets service manages wallet operations.

## Methods

### `getWallets()`

Get all wallets for your account.

```typescript
const wallets = await afriex.wallets.getWallets();
```

### `getWallet(currency)`

Get a specific wallet by currency.

```typescript
const wallet = await afriex.wallets.getWallet('USD');
```

### `getBalance(currency)`

Get the balance for a specific currency.

```typescript
const balance = await afriex.wallets.getBalance('NGN');
console.log(`Available: ${balance.available}, Pending: ${balance.pending}`);
```

### `getTransactions(params?)`

Get wallet transaction history.

```typescript
const transactions = await afriex.wallets.getTransactions({
  currency: 'USD',
  limit: 50,
  offset: 0,
});
```
